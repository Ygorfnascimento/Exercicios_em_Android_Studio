# 📱 Exercicios em Android Studio

Este repositório contém o mapeamento, a estruturação cronológica e os scripts de implementação prática para o ecossistema de aprendizado e consolidação de desenvolvimento mobile nativo utilizando o Android Studio e a linguagem Kotlin.

---

## 🎯 Objetivo

O propósito deste projeto é estruturar a arquitetura de aprendizado necessária para gerenciar a evolução do desenvolvimento mobile, garantindo a fixação de conceitos de layouts em XML, ciclo de vida de Activities, regras de negócio isoladas e manipulação de recursos visuais dinâmicos através de desafios práticos organizados cronologicamente.

---

## 📂 Estrutura do Repositório

O projeto combina a organização temporal e modular de laboratórios práticos focados no ecossistema Android:

### 📅 Organização por Períodos Mensais
Os módulos estão divididos para refletir a evolução contínua em arquitetura mobile e componentização:

* **Fev_2026 (Projeto_Login & Projeto_Android):** Mapeamento e fluxo de autenticação com isolamento de regras de negócios (`UserBusiness`), gerenciamento de escopo global (`AppContent`) e controle estrutural entre telas com fluxo de intenções (`MainActivity` e `HomeActivity`). Inclui o design estruturado de componentes visuais básicos com ativos estáticos personalizados.
* **Abr_2026 (Projeto - Caf):** Construção de dashboards e interfaces modernas orientadas à experiência do usuário (UX), explorando a renderização e o gerenciamento de vetores customizados (`Vector Drawables`).

## 🚀 Como Executar o Projeto Localmente

1. **Clonagem do Repositório:** Abra o seu terminal e execute o comando abaixo para clonar o repositório:
   ```bash
   git clone https://github.com/Ygorfnascimento/Exercicios_em_Android_Studio.git)
   ```
2. **Abertura do Projeto:** Inicialize o Android Studio, selecione a opção Open e aponte para a pasta raiz do módulo mensal específico que deseja executar.
3. **Sincronização e Execução:** Aguarde a indexação e o Gradle Sync baixarem as dependências. Conecte um emulador ou dispositivo físico e pressione Shift + F10 (ou o botão de play) para compilar.
